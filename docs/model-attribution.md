# Model attribution

Upright bundles the MediaPipe Pose Landmarker Lite model for local, one-person pose estimation.

- Runtime package: `@mediapipe/tasks-vision@0.10.35`
- Bundled asset: `src/renderer/public/models/pose_landmarker_lite.task`
- Exact model source: `https://storage.googleapis.com/mediapipe-models/pose_landmarker/pose_landmarker_lite/float16/1/pose_landmarker_lite.task`
- SHA-256: `59929e1d1ee95287735ddd833b19cf4ac46d29bc7afddbbf6753c459690d574a`
- Source project: [Google AI Edge MediaPipe](https://github.com/google-ai-edge/mediapipe)
- Documentation: [Pose Landmarker](https://developers.google.com/mediapipe/solutions/vision/pose_landmarker)
- Framework license: Apache-2.0
- Model asset terms: Apache License 2.0, distributed with the Google MediaPipe model bundle

The model and WebAssembly runtime are packaged with Upright. Camera frames are processed locally and are not retained by the application.
